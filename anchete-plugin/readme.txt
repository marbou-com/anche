== Description ==

This plugin provides the ability to conduct simple surveys during articles.
It is possible by registering the content (title and answer items) you want to survey and embedding the created tag in the article.

The user who browses the article answers the questionnaire in the article by pressing a button.
By pressing the button, you can see the survey results.
It is not managed by sessions or cookies.

The number of survey responses is saved for each survey in a table created when the survey plugin is installed.

Once created, the title, items, and number of responses cannot be changed.
If it is necessary to make changes, unpublish or delete the relevant questionnaire from the questionnaire management and create a new questionnaire.
Also use the newly created tag.

== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “Easy Anchete”.
3. Select the “Easy Anchete” plugin and click “Install”.
4. Activate the plugin.

OR…

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click “Plugins” -> “Add new”.
4. Click “upload”.. then browse to the .zip file downloaded from this page.
5. Click “Install”.. and then “Activate plugin”.
